# 🌐 Interactive Responsive Webpage with JavaScript

## 📌 Project Overview

This project is a complete interactive webpage built using **HTML, CSS, and JavaScript**. It demonstrates how to structure a webpage, apply external styling, make it responsive, and add basic interactivity using JavaScript.

---

## ✨ Features

* Clean and structured HTML layout
* External CSS for styling and design
* Responsive design using media queries
* Styled image with borders and rounded corners
* Button to visit an external profile
* Interactive button to change background color dynamically

---

## 🛠️ Technologies Used

* **HTML5** – for webpage structure
* **CSS3** – for styling and responsive design
* **JavaScript** – for interactivity (background color change)

---

## 📂 Project Structure

```id="h7rj8u"
project-folder/
│
├── index.html     # Main HTML file
├── styles.css     # External CSS file
└── script.js      # JavaScript file
```

---

## 🚀 How to Run the Project

1. Create a folder on your system
2. Add the following files:

   * `index.html`
   * `styles.css`
   * `script.js`
3. Make sure all files are in the same folder
4. Open `index.html` in any web browser

---

## 📱 Responsive Design

The webpage adjusts based on screen size:

* **Desktop** → Full layout
* **Tablet (≤768px)** → Adjusted font and image size
* **Mobile (≤480px)** → Smaller text and buttons

---

## ⚡ Functionality

* Clicking the **“Visit My Profile”** button opens a LinkedIn profile in a new tab
* Clicking the **“Change Background Color”** button changes the background color randomly using JavaScript

---

## 🎯 Learning Outcomes
